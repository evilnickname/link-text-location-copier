About

Link Text and Location Copier is a Firefox WebExtension that helps you creating links right from the context menu. It copies a link’s text and location, the page title and URL, or selected text and page url as either plain text, HTML, Markdown or BB Code links.

Download Link Text and Location Copier 1.5.0

  • Requires: Firefox 56+

It is heavily inspired by CoLT, one of my favorite add-ons, which will not be ported for Firefox 57. It is also my first attempt at creating a web extension, so I blatantly stole some code to make it work.

Known issues

  • Before Firefox 56 the link title will always be “undefined”.
  • Before Firefox 56 text selection is limited to 150 characters.
  • If you have set the about:config preference dom.event.clipboardevents.enabled to false, this add-on will not work.
  • Link Text and Location Copier will not work on XML documents, as these do not support document.execCommand(copy).

Custom Format Documentation

Easy as pie: with the variables listed below, create whatever format you need.

Context dependant custom format variables
(Deprecated. Will probably be removed in version 2.)
Variable Link Page Selection Image inside a link
%T Link text Page title Selected text 1 All text inside the link 1
%U Link URL Page URL Page URL Link URL
%I Image URL
Context independent custom format variables
Variable Value
%linktitle% Link title 2
%linkurl% Link URL 2
%pagetitle% Page title
%pageurl% Page URL
%selection% Selected text 2
%N Newline character 3
%B Tab character
%L Localized time stamp
  1. Including an image’s alt-attribute
  2. Will return an empty string when unavailable.
  3. \n. For HTML, you can insert <br /> into the template.

Options

  • Don’t encode link location

    Firefox encodes non-ANSI characters (Cyrillic, Greek, Vietnamese, what-have-you) in URLs by default when copying. Links like https://ru.wikipedia.org/wiki/Браузер get turned into https://ru.wikipedia.org/wiki/%D0%91%D1%80%D0%B0%D1%83%D0%B7%D0%B5%D1%80 when pasted.

    To prevent this, turn this feature on and save settings.

  • Don’t show context menu prefix

    By default, Link Text and Location Copier shows a “Copy … as ” prefix in te context menu before the menu label you set as custom format. This option allows you to overide this behavior, and uses the menu label as is in the context menu.
